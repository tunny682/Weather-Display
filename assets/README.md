# Assets

## Weather icons

Place **PNG** weather icons in this folder:

```
assets/
  icons/
    sun.png      ← clear, mainly_clear
    cloud.png    ← partly_cloudy, overcast
    rain.png     ← rain, drizzle
    snow.png     ← snow
    storm.png    ← thunderstorm
    fog.png      ← fog
```

- Use **transparent background** so they look good on the dark display.
- Suggested size: **64×64** or **128×128** px (the app scales them).
- File names must match exactly (lowercase) or the app falls back to drawn shapes.

If a file is missing, the app draws a simple shape instead. You can add icons gradually.
